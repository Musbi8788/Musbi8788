### [AttendanceGM](https://www.attendancegm.com)
Workforce attendance evolving into a complete HR management platform.
- Replaces paper-based tracking with offline-first digital workflows
- Features attendance, leave management, payroll integration, and staff document handling
- Deployed with active organizational clients including WAVE Gambia and Sal Consult

Real-time emergency fire reporting and response system.

## Mission

Technology in Africa shouldn't be adapted from elsewhere — it should be built here, by people who understand the problems firsthand.

I build because manual systems still govern payroll, emergency response still relies on phone calls, and healthcare scheduling still happens on paper. These aren't just inconveniences — they're structural inefficiencies that hold back entire economies.

Every system I deploy replaces friction with function. AttendanceGM started as an attendance tracker. Today it's evolving into a complete HR platform because I listen to the organizations using it — not because a roadmap written in Silicon Valley said so.
